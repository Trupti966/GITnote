To initiate git in work folder --------> git init

To check git status ------> git status

To create a file in work folder ----> touch fileName.extension

To make tracked file by git -----> git add fileName.extension

To track all file -----> git add -A

To commit -----> git commit -m " commit message "

To get all information about folder/ file's present inside the work folder -------> ls -lars   or ls -lart

To get back my deleted files ----> git checkout fileName.extension
